ntfy web-push

To get started, install ntfy-webpush and run it to generate keys, then follow the link then the output to reload this page with a public key. $ sudo pip install ntfy-webpush $ ntfy-webpush ....

When a key is loaded, click subscribe to subscribe to notifications in this browser and then copy the config below into your ntfy.yml

Sorry, your browser doesn't support web-push.

Then send a test notification with:

ntfy -b ntfy_webpush send 'testing webpush!'
